## Using the glossary vs. the spec index

- **Glossary** — look up a term the narrative uses (e.g. "path hash", "flood advert", "ECDH shared secret"). Each entry gives the concept and points to the page where it is explained in depth.
- **Spec index** — find the right `docs.meshcore.nz` page when you need the authoritative byte-level detail (frame layout, exact command codes, payload fields). The guide's cross-link policy is: narrative here, bytes on `docs.meshcore.nz`.
